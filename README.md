# 🚀 Log Monitoring SaaS Dashboard

A full-stack log monitoring system built using FastAPI and JavaScript.

## 🔥 Features
- Real-time log streaming
- Search & filtering
- Dashboard with stats
- Alerts panel
- Modern SaaS UI

## 🛠 Tech Stack
- FastAPI (Backend)
- SQLite (Database)
- HTML, CSS, JavaScript (Frontend)

## ▶️ How to Run

```bash
uvicorn main:app --reload
